# Баг‑репорты UI ToDoMVC
Набор найденных дефектов в приложении ToDoMVC (JavaScript ES5). Оформлены в формате, приближенном к корпоративным стандартам.


BUG_ES5_001 — Пустая задача создаётся при вводе пробелов
Severity: Major
Priority: P1
Environment: ToDoMVC JavaScript ES5, Chrome 123
Steps:
Ввести " " в поле ввода
Нажать Enter
Actual:  
Создаётся пустая задача
Expected:  
Задача не должна создаваться


BUG_ES5_002 — Редактирование позволяет сохранить пустой текст
Severity: Major
Priority: P2
Steps:
Добавить задачу
Двойной клик → редактирование
Удалить текст
Нажать Enter
Actual:  
Создаётся пустая задача
Expected:  
Пустая задача должна удаляться или редактирование отменяться


BUG_ES5_003 — Фильтр Active отображает выполненные задачи
Severity: Major
Priority: P2
Steps:
Добавить 3 задачи
Отметить 1 как выполненную
Нажать Active
Actual:  
Выполненная задача отображается
Expected:  
Должны отображаться только активные


BUG_ES5_004 — Clear completed удаляет активные задачи при быстром клике
Severity: Critical
Priority: P1
Steps:
Добавить 3 задачи
Отметить 1 как выполненную
Быстро нажать Clear completed 2–3 раза
Actual:  
Удаляются активные задачи
Expected:  
Удаляться должны только выполненные


BUG_ES5_005 — Подсветка активного фильтра сбрасывается после обновления страницы
Severity: Minor
Priority: P3
Steps:
Нажать Completed
Обновить страницу
Actual:  
Подсветка возвращается на All
Expected:  
Подсветка должна соответствовать выбранному фильтру


BUG_ES5_006 — Кнопка удаления не отображается при быстром наведении
Severity: Minor
Priority: P3
Steps:
Добавить задачу
Быстро провести курсором по задаче
Actual:  
Кнопка удаления иногда не появляется
Expected:  
Кнопка должна отображаться всегда при наведении


BUG_ES5_007 — Toggle All не всегда корректно снимает выполнение
Severity: Medium
Priority: P2
Steps:
Добавить 3 задачи
Нажать Toggle All
Нажать Toggle All снова
Actual:  
Одна из задач остаётся выполненной
Expected:  
Все задачи должны стать активными
