# UI ToDoMVC — Test Cases (Priority P1)

Набор тест‑кейсов уровня Middle для проверки критического UI‑функционала ToDoMVC.

---

## TC_UI_001 — Добавление новой задачи
**Приоритет:** P1  
**Предусловия:** Страница ToDoMVC открыта

### Шаги:
1. Ввести текст «Test task» в поле ввода новой задачи  
2. Нажать Enter  

### Ожидаемый результат:
- В списке появляется новая задача с текстом «Test task»  
- Поле ввода очищается  
- Счётчик задач увеличивается на 1  

---

## TC_UI_002 — Добавление пустой задачи (негативный)
**Приоритет:** P1  
**Предусловия:** Страница ToDoMVC открыта

### Шаги:
1. Оставить поле ввода пустым  
2. Нажать Enter  

### Ожидаемый результат:
- Новая задача не создаётся  
- Список задач не изменяется  
- Счётчик задач остаётся прежним  

---

## TC_UI_003 — Отметка задачи как выполненной
**Приоритет:** P1  
**Предусловия:** Есть хотя бы одна активная задача

### Шаги:
1. Нажать на чекбокс слева от задачи  

### Ожидаемый результат:
- Задача получает статус «выполнена»  
- Текст задачи отображается зачёркнутым  
- Счётчик активных задач уменьшается на 1  

---

## TC_UI_004 — Снятие отметки выполнения
**Приоритет:** P1  
**Предусловия:** Есть хотя бы одна выполненная задача

### Шаги:
1. Нажать на чекбокс выполненной задачи  

### Ожидаемый результат:
- Задача возвращается в статус «активная»  
- Зачёркивание исчезает  
- Счётчик активных задач увеличивается на 1  

---

## TC_UI_005 — Удаление задачи
**Приоритет:** P1  
**Предусловия:** Есть хотя бы одна задача

### Шаги:
1. Навести курсор на задачу  
2. Нажать кнопку удаления (крестик)

### Ожидаемый результат:
- Задача исчезает из списка  
- Счётчик задач уменьшается на 1  

---

## TC_UI_006 — Редактирование задачи
**Приоритет:** P1  
**Предусловия:** Есть хотя бы одна задача

### Шаги:
1. Дважды кликнуть по тексту задачи  
2. Ввести новый текст «Updated task»  
3. Нажать Enter  

### Ожидаемый результат:
- Текст задачи обновляется на «Updated task»  
- Режим редактирования закрывается  

---

## TC_UI_007 — Очистка завершённых задач
**Приоритет:** P1  
**Предусловия:** Есть хотя бы одна выполненная задача

### Шаги:
1. Нажать кнопку «Clear completed»

### Ожидаемый результат:
- Все выполненные задачи удаляются  
- В списке остаются только активные  
- Кнопка «Clear completed» исчезает, если выполненных задач больше нет
