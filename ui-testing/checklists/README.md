# UI Checklists
Здесь будут храниться чек-листы для UI.

# Чек-лист UI ToDoMVC

## P1 — Критический функционал

### Добавление задач
- [ ] Поле ввода отображается
- [ ] Можно ввести текст задачи
- [ ] Задача добавляется по Enter
- [ ] Новая задача появляется в списке
- [ ] Текст задачи отображается корректно
- [ ] После добавления поле ввода очищается

### Выполнение / снятие выполнения
- [ ] Чекбокс задачи отображается
- [ ] Можно отметить задачу как выполненную
- [ ] Выполненная задача отображается зачёркнутой
- [ ] Можно снять отметку выполнения
- [ ] Стиль задачи возвращается к обычному

### Удаление задач
- [ ] Кнопка удаления отображается при наведении
- [ ] Задача удаляется по нажатию на кнопку
- [ ] Удалённая задача исчезает из списка
- [ ] Удаление не влияет на другие задачи

### Список задач
- [ ] Список отображается при наличии задач
- [ ] Пустой список скрывает блок задач
- [ ] Порядок задач сохраняется после обновления страницы

### Счётчик задач
- [ ] Счётчик отображается
- [ ] Счётчик уменьшается при выполнении задачи
- [ ] Счётчик увеличивается при снятии выполнения
- [ ] Счётчик корректен при удалении задач

---

## P2 — Основной функционал

### Редактирование задач
- [ ] Двойной клик переводит задачу в режим редактирования
- [ ] Можно изменить текст задачи
- [ ] Enter сохраняет изменения
- [ ] Esc отменяет изменения
- [ ] Поле редактирования исчезает после сохранения

### Фильтры (All / Active / Completed)
- [ ] Фильтр All отображает все задачи
- [ ] Фильтр Active отображает только активные
- [ ] Фильтр Completed отображает только выполненные
- [ ] Переключение фильтров не сбрасывает состояние задач
- [ ] Активный фильтр подсвечивается

### Clear completed
- [ ] Кнопка отображается только при наличии выполненных задач
- [ ] По нажатию удаляются только выполненные задачи
- [ ] Активные задачи остаются

---

## P3 — Негативные сценарии и edge cases

### Негативные сценарии
- [ ] Пустая задача не добавляется
- [ ] Задача из пробелов не добавляется
- [ ] Очень длинная задача отображается корректно
- [ ] HTML-теги отображаются как текст
- [ ] Спецсимволы отображаются корректно

### Edge cases
- [ ] Приложение работает при 0 задач
- [ ] Приложение работает при 1 задаче
- [ ] Приложение работает при 100+ задачах
- [ ] Все задачи выполнены — фильтры работают корректно
- [ ] Все задачи активны — фильтры работают корректно
